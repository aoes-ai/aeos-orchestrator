# AEOS Orchestrator

## Autonomous Engineering Operating System

Complete free, open-source engineering platform controlled entirely from iPhone.

### Architecture

- **Orchestration**: AIOS + Omnigent + GoiabaSeeds
- **Greenfield Builders**: Singulary, December, Genesis, WebCraft, Micracode, Tinykit
- **Coding Agents**: OpenHands, Goose, Aider, Cline, OpenCode, Pi, Nanocoder, gptme
- **Error Detection**: Ultimate Bug Scanner, yobitsugi, CodeMedic, Undo AI, Semgrep, CodeQL
- **Multi-Agent Orchestration**: CrewAI, AG2, OpenForgeAI, Synapse, Conductor
- **MCP Servers**: GitHub, Filesystem, PostgreSQL, Redis, Chroma, Semgrep, Trivy
- **Security**: Gitleaks, Trivy, Semgrep, OWASP Dependency-Check
- **Testing**: Vitest, Playwright, Supertest
- **iPhone Control**: CodeAgents Mobile + GoiabaSeeds (Telegram)

### Commands

| Command | Purpose |
|---------|---------|
| `/add-project --name X --repo Y` | Add a new project |
| `/list-projects` | List all projects |
| `/run audit --project X` | Audit a project |
| `/run audit --repo URL` | Audit any repository |
| `/run fix --project X` | Fix a project |
| `/run build --name X --desc Y` | Build from scratch |
| `/run deploy --project X` | Deploy a project |
| `/status` | Check status |
| `/stop` | Stop running task |

### Getting Started

1. Install CodeAgents Mobile on iPhone
2. Pair with Telegram via GoiabaSeeds
3. Add your first project
4. Run an audit

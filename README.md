# AEOS Orchestrator

## Autonomous Engineering Operating System

Complete free, open-source engineering platform controlled entirely from iPhone.

### Architecture
- **Orchestrator**: Wallfacer + Quay + Omnigent
- **Memory**: free-context-hub (persistent lessons, guardrails)
- **Code Intelligence**: Nexus-MCP (code graph, impact analysis)
- **Multi-Agent**: Multi-Agent MCP (Git-sandboxed)
- **Web Research**: Hound MCP (fetch, crawl, search)
- **AI Workspace**: Odyssey (chat, agents, research)
- **Agents**: OpenHands, Goose, Aider, Cline, Roo Code, OpenCode, Pi, Nanocoder

### Target Projects
- World-Bank (primary)
- Additional projects via GitMCP

### iPhone Control
- CodeAgents Mobile
- GoiabaSeeds (Telegram)
